Download vs Size Analysis

If you look at this bar chart, you will notice how the most-downloaded App Store applications are in the range of 60-80 megabytes in terms of app file size. The analysis which gives us these figures is influenced by Facebook and YouTube, two applications with more than 1 billion downloads: there is a correlation between app file size and downloads (apps with a large number of downloads tend to be large in file size, too). On the left, where there is the ratings range, the range 80-100 megabytes stands out with an average rating a little higher than 4.0
